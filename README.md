# vibe-check

Production-readiness skill for Claude. Reviews FastAPI + React apps before deployment — checks secrets hygiene, data model health, deploy config, and frontend/backend boundary concerns.

## Usage

Claude picks this up automatically when you say things like:
- "Is this ready to ship?"
- "Sanity check my app before deploy"
- "Pre-launch review"

## Stack

Targets: FastAPI · React · Vercel
